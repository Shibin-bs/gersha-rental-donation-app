# Gersha – Rental & Donation Platform

A community-driven mobile application that allows users to **list, rent, and donate items locally**.
The platform encourages resource sharing, reduces waste, and helps communities access useful items without needing to purchase them.

## Features

* List items for **rent or donation**
* Browse items available in the local community
* Item details with descriptions and images
* Simple and clean mobile interface
* Community-driven sharing system

## Tech Stack

* **Flutter** – Cross-platform mobile app development
* **Dart** – Application logic
* **Firebase / Backend services** (if used)

## Getting Started

Clone the repository:

```
git clone https://github.com/Shibin-bs/gersha-rental-donation-app.git
```

Navigate into the project folder:

```
cd gersha-rental-donation-app
```

Install dependencies:

```
flutter pub get
```

Run the application:

```
flutter run
```

## Purpose

The goal of Gersha is to **promote community sharing** by making it easy for people to rent or donate items they no longer use.
This reduces unnecessary purchases and supports a more sustainable lifestyle.
